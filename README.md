# Swift Hash&Convert

####Hash app for Mac for develop use, written in swift.

---
####Support Hash Algorithm:
>
1. MD5
2. SHA1
3. SHA224
4. SHA256
5. SHA384
6. SHA512
7. CRC32

####Support Convert Type:
>
1. Decimal to Hex String
2. Hex to Decimal String
3. String to Base64
4. Base64 to String

####Source Input:
>
1. File, you can drop your file to the drop zone
2. String by input
